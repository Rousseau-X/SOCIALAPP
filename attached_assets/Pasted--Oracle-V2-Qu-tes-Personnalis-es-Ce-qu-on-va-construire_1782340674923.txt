🧠 Oracle V2 — Quêtes Personnalisées

✅ Ce qu'on va construire

L’Oracle va analyser l’activité de chaque utilisateur et générer chaque jour une quête personnalisée.

---

📋 Plan d'attaque

1. Ce que l'Oracle fait chaque jour

· Analyse l'activité récente (posts, messages, likes, connexions)
· Génère une mission adaptée au profil
· Attribue des récompenses (XP + coins)
· Expire après 24h

2. Exemple de quêtes

Utilisateur Mission Récompense
Actif (3+ posts/jour) "Publie une photo aujourd'hui" 50 XP + 10 coins
Nouveau (0 post) "Fais ton premier post" 100 XP + 20 coins
Peu de messages "Envoie 5 messages" 30 XP + 5 coins
Très actif en likes "Liker 10 posts" 40 XP + 8 coins
Groupe inactif "Rejoins un groupe" 50 XP + 15 coins
Story inactive "Publie une story" 60 XP + 12 coins

🚀 Exemple de code pour générer une quête

```javascript
// lib/oracle.js
function generateQuest(user) {
    const actions = [
        { type: 'post', condition: user.postsToday < 1, reward: { xp: 50, coins: 10 }, text: 'Publie un post aujourd\'hui' },
        { type: 'message', condition: user.messagesToday < 5, reward: { xp: 30, coins: 5 }, text: 'Envoie 5 messages' },
        { type: 'story', condition: user.storiesToday < 1, reward: { xp: 40, coins: 8 }, text: 'Publie une story aujourd\'hui' },
        { type: 'like', condition: user.likesToday < 10, reward: { xp: 25, coins: 5 }, text: 'Liker 10 publications' },
        { type: 'ai', condition: user.aiToday < 1, reward: { xp: 20, coins: 3 }, text: 'Utilise l\'IA une fois' },
    ];

    const available = actions.filter(a => a.condition);
    if (available.length === 0) {
        return {
            text: 'Félicitations, tu es déjà très actif ! Continue comme ça.',
            reward: { xp: 10, coins: 5 },
        };
    }
    const quest = available[Math.floor(Math.random() * available.length)];
    return {
        text: quest.text,
        reward: quest.reward,
        type: quest.type,
    };
}
```
